## Description

name_a10フォルダ以下に元データがあります。

`sei.json`は姓のデータです。`mei.json`は名のデータです。  

scriptsフォルダ以下には人名データを抜き出してJSONに整形するスクリプト（Python）があります。  
実行はPython3環境が必要です。

姓であれば以下のとおりです。  
```
python jinmei_name_a10.py '~/name_a10/A10_SEI.TXT' './sei.json'
```
